# Finanças Pessoais (v0)

Este repositório tem dois READMEs, com públicos diferentes:

- `README_USER.md` — guia para uso do app (passo a passo + formatos de CSV por cartão).
- `README_TECHNICAL.md` — documentação técnica voltada para IA (arquitetura, fluxo de dados e **row_hash**/dedupe).

Quickstart:

```bash
docker compose up --build
```

Depois acesse `http://localhost:8501`.
